¿Cómo funciona?

Este proyecto utiliza un modelo de regresión lineal entrenado con datos sintéticos para predecir el Índice de Masa Corporal (IMC).

El modelo fue implementado usando TensorFlow.js y se ejecuta directamente en tu navegador.
